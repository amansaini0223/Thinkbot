# 📖 About Tech AI

**Tech AI** is a next-generation AI chatbot designed to deliver fast, intelligent, and seamless conversations using Groq's ultra-low latency Large Language Models.

Built with a clean ChatGPT-inspired interface, Tech AI focuses on speed, usability, and a premium user experience while providing customizable AI personalities, streaming responses, prompt enhancement, chat analytics, and export capabilities.

Whether you're writing code, brainstorming ideas, analyzing data, or simply asking questions, Tech AI provides a smooth and responsive AI experience.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/your-username/Tech-AI.git
```

## Move into Project

```bash
cd Tech-AI
```

## Create Virtual Environment

```bash
python -m venv .venv
```

## Activate Environment

### Windows

```bash
.venv\Scripts\activate
```

### Linux / macOS

```bash
source .venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Configure Environment

Create a `.env` file.

```env
GROQ_API_KEY=YOUR_API_KEY
```

---

## Run Application

```bash
streamlit run app.py
```
