Project Name:
THINKBOT AI

Description:
THINKBOT AI is an advanced AI chatbot built using Streamlit and Groq API.
It provides real-time streaming responses with multiple AI personas,
chat memory, prompt enhancement, analytics, export options, and a premium UI.

=========================================
вҳҒ Deploy on Streamlit Cloud
=========================================

1. Login to Streamlit Cloud

2. Click New App

3. Select GitHub Repository

4. Select app.py

5. Open Advanced Settings

6. Add Secret

GROQ_API_KEY="YOUR_API_KEY"

7. Click Deploy
